**一键配置【NOFX】AI交易系统**

*本程序仅支持 Ubuntu24.04系统*

```
wget https://raw.githubusercontent.com/ShenXuGongZi/easyNOFX/refs/heads/main/install_nofx_ubuntu.sh  && chmod +x install_nofx_ubuntu.sh && bash install_nofx_ubuntu.sh 
```

**管理脚本**

```
nofx
```

功能包括：

* 查看服务状态
* 启动/停止/重启服务
* 查看实时日志
* 编辑配置文件
* 备份配置
* 资源监控
* 完全卸载
